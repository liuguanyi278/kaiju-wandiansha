## v1.0 首次部署上线

日期：2026-07-06

### 部署平台

Vercel

### 线上链接

https://kaiju-wandiansha.vercel.app/

### GitHub 仓库

https://github.com/liuguanyi278/kaiju-wandiansha

### 部署方式

通过 GitHub 仓库导入 Vercel，由 Vercel 自动识别 Vite 项目并完成构建部署。

### 构建配置

* Framework Preset：Vite
* Install Command：npm install
* Build Command：npm run build
* Output Directory：dist

* 首页
* 游戏大厅
* 真心话大冒险
* 转盘惩罚
* 默契问答
* 移动端优先布局
* 本地静态题库
* 卡牌翻转放大交互
* 转盘旋转交互
* 随机换题交互

### 线上验收结果

* 首页可正常访问
* 游戏大厅可正常进入
* 真心话大冒险可正常翻牌和换题
* 转盘惩罚可正常旋转并展示结果
* 默契问答可正常切换问题
* 返回大厅逻辑正常
* 手机端可正常打开
* 朋友外部访问验证通过

### 结论

《开局玩点啥》V1.0 已完成从本地 MVP 到 GitHub 仓库，再到 Vercel 线上部署的完整流程，具备可访问、可展示、可继续迭代的项目基础。
