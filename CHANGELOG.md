## v1.0.1 - 游戏大厅宫格化优化

日期：2026-07-06

### 本次目标
优化游戏大厅的信息展示方式，让页面更符合“小游戏合集”的产品定位，方便用户一览所有游戏。

### 修改内容
- 给游戏配置增加 icon 字段。
- 游戏大厅从纵向大卡片列表改为 3 列宫格入口。
- 移除游戏大厅中的简介、标签和“开始”按钮。
- 点击整个游戏入口即可进入对应游戏。
- 将游戏简介、标签和玩法说明移动到具体游戏页面顶部。
- 增加极窄屏 2 列适配。
- 保留真心话大冒险、转盘惩罚、默契问答原有核心逻辑。

### 修改原因
原游戏大厅卡片过大，用户一屏只能看到少量游戏，不利于后续扩展更多玩法。宫格入口更适合“小游戏合集”产品形态。

### 测试结果
- npm.cmd run build 通过。
- TypeScript 编译成功。
- Vite 生产构建成功。
- 三个游戏核心交互未受影响。

### 下一步计划
- 继续检查手机端真实显示效果。
- 根据体验决定是否优化游戏页面顶部信息密度。
